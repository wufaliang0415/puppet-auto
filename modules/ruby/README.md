What is it?
===========

A Puppet module that manages rubygems to resolve the issue where multiple
modules need the same gem.

Released under the Apache 2.0 licence

Usage:
------

To install:
<pre>
  include ruby
</pre>

To include an additional gem:
<pre>
  include ruby::httparty
</pre>
